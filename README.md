# Execute in development environment

1. Clone the repository
2. Execute 
 ```
    yarn install
 ```
3. Nest CLI must to be installed
```
npm i -g @nestjs/cli
```
4. Raise the database
```
docker-compose up -d
```

## Used Stack
* MongoDB
* Nest
